# E-Commerce Sales Analysis Using SQL

## Project Overview

This project focuses on analyzing e-commerce sales data using SQL to derive meaningful business insights. The analysis helps understand customer purchasing behavior, sales performance, and revenue contribution across different product categories.

## Objectives

* Analyze customer spending patterns
* Identify top-performing customers
* Calculate Average Order Value (AOV)
* Measure category-wise revenue contribution
* Rank customers based on total spending
* Generate actionable business insights from sales data

## Tools Used

* MySQL
* SQL
* MySQL Workbench

## Database Structure

The database consists of the following tables:

* Customers
* Orders
* Products
* Categories
* Order Details

These tables are connected using primary and foreign key relationships.

## SQL Concepts Applied

* SELECT Statements
* INNER JOIN
* Aggregate Functions (SUM, COUNT, AVG)
* GROUP BY
* ORDER BY
* LIMIT
* Window Functions (RANK)
* Subqueries
* Primary Keys & Foreign Keys

## Analysis Performed

### Customer Analysis

* Total spending by each customer
* Total orders placed by customers
* Ranking customers based on spending

### Sales Analysis

* Total revenue generated
* Total units sold
* Average Order Value (AOV)

### Category Analysis

* Revenue generated by each category
* Percentage contribution of each category to total revenue

## Key Insights

* Identified the highest-spending customers
* Measured overall business revenue
* Calculated Average Order Value (AOV)
* Ranked customers using SQL Window Functions
* Analyzed category-wise revenue contribution
